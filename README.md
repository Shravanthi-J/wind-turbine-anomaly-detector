# turbine-anomaly-detector

An end to end Machine Learning Solution for reducing the down time of a turbine using a modular, reproducible pipeline built with Kedro.

## Key Objectives
* Build a reproducible ML pipeline using Kedro
* Perform structured preprocessing of multi-year turbine data
* Modeling approach formulation and baseline model development and selection
* Hyperparameter Tuning, Experiment Tracking and Model Registry

## Tech Stack
* Python
* Kedro
* Pandas & NumPy (Data Processing)
* Matplotlib / Seaborn (Visualization)
* Jupyter Notebooks
* Git (Version Control)
* uv (Environment & Dependency Management)

## How to run your Kedro pipeline

You can run your Kedro project with:

```
kedro run
```
